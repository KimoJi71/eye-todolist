# eye-todolist

### 下載程式碼
```
git clone https://github.com/KimoJi71/eye-todolist.git
```

### 下載套件
```
yarn install
```

### 編譯程式、啟動 Server
```
yarn serve
```

### 輸入網址
```
http://localhost:80804
```
